# Coffee Machine Simulator

This is a simple interactive coffee machine simulator built using **JavaScript**, **HTML**, and **CSS**. It allows users to manage ingredients and credit, and make different types of coffee.

## Features

- Display current **credit**, **water**, **coffee**, and **milk** levels.
- Add **credit**, **water**, **coffee**, and **milk** to the machine.
- Make three types of coffee:
  - Short Espresso
  - Long Espresso
  - Capuchino
- Real-time status updates on the screen.
- Prevents making coffee if ingredients or credit are insufficient.

## How to Use

1. Open `index.html` in a browser.
2. Check current status on the machine screen.
3. Click **Add Credit**, **Add Water**, **Add Coffee**, or **Add Milk** to top up ingredients.
4. Choose the coffee type by clicking the corresponding button.
5. The machine will prepare the coffee and update the status automatically.
